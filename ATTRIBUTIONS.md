## Recommendations before any wider publish

1. Strip EXIF metadata from all images to remove any embedded photographer names, GPS coordinates, or camera identifiers. `exiftool -all= images/*` handles this in one command.
2. Confirm the licence status of each Wikimedia image against its current Commons file page.
3. Where the university has an updated official portrait, replace the placeholder from gbpuat.ac.in.
4. If any photograph originates from a third-party photographer contracted by the university, add explicit credit here.
